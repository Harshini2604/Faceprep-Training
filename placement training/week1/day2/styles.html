FRONTEND DEVELOPMENT

Front-end development is the process of building components that interact with users. Front-end is the part that is accessible to the user only, It should be self-explanatory and must be user-friendly and designed. so to achieve this we have some basic languages which can be used to create interactive web pages.

Examples: User interface, buttons, user-entered data, websites, and user experience (UX) features.

Frameworks provide extra features, structure, fast reloading, and many more things so that developers can develop a SPA(Single page application) easily.Frameworks add more features to their respective languages. Some of them provide inbuilt functions or pre-defined components so that developer can directly use those in their projects
